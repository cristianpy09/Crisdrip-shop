# CrisDripShop

Modern online store built with **Next.js**, **TypeScript**, and **Tailwind CSS**.  
Includes product views, size and quantity selectors, a shopping cart, and a fully responsive design.

## Technologies
- Next.js 14  
- React 18  
- Tailwind CSS  
- TypeScript  
- Swiper.js  

## Run Locally
```bash
npm install
npm run dev
```
