#  CrisDripShop

Tienda online moderna desarrollada con **Next.js**, **TypeScript** y **Tailwind CSS**.  
Incluye vista de productos, selección de tallas y cantidades, carrito de compras y diseño 100% responsive.

##  Tecnologías
- Next.js 14  
- React 18  
- Tailwind CSS  
- TypeScript  
- Swiper.js  

##  Ejecutar localmente
```bash
npm install
npm run dev
```
